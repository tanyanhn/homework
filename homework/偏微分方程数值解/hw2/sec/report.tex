\subsection{问题}
在初始条件 
\[f_j = \left\{ 
\begin{aligned}
  x_j \qquad         &0 \leq x \leq \pi \\
  2 \pi - x_j \qquad &\pi \leq x \leq 2\pi 
\end{aligned}
 \right. \]
 下, 使用Forward-Euler, Lax-Friedrichs, Lax-Wendroff方法
 数值求解 
 \[ u_t = u_x \]

\subsection{准确解}
使用$h = 2 * \pi / 5000, k = h / 2$ Lax-Wendroff 方法的数值模拟结果作为精确解.
